Armbian编译和配置NeoVim

  • 本文编写于2023年7月5日
  • NeoVim版本：v0.9.1
  • 编译完好像要占挺大空间的
  • 真服了，编译3次了才发现可以用Snap下载
sudo snap install nvim
# 版本是最新的v0.9.1
sudo apt install neovim
# 版本是v0.6.1-3

从GitHub下载

  • Gitee上也有代码镜像，但是一般都比较旧
  • 最好用git clone，下载压缩包好像会有问题
  • 如果下载的是压缩包，是用不了git checkout stable的
  • 那样的话，貌似编译时会在git checkout stable卡住

安装依赖​

sudo apt install ninja-build gettext libtool libtool-bin cmake g++ pkg-config unzip curl doxygen

修改文件​

如果你能流畅访问GitHub或者有魔法、梯子什么的，可以跳过这一步

  1. 编辑在NeoVim项目文件夹的下cmake.deps/CmakeList.txt
  2. 把134~197行中的http链接的文件全部用wget下载下来
  3. 然后把http链接全部替换成file链接

比如：

https://github.com/neovim/unibilium/archive/d72c3598e7ac5d1ebf86ee268b8b4ed95c0fa628.tar.gz

改成：

file://home/用户名/Download/d72c3598e7ac5d1ebf86ee268b8b4ed95c0fa628.tar.gz
  • 好像不能直接用~来表示用户文件夹
  • 不要放在什么乱七八糟的文件夹下，像什么中文文件夹，根目录之类的
  • 如果你用wget也不能下载文件，可以在Gitee上搜索一下有没有镜像，或者从GitHub导入

编译​

make CMAKE_BUILD_TYPE=RelWithDebInfo

安装​

sudo make install

速度很慢，耐心等待

配置​

推荐B站@蛋老师分享的配置

开始使用NeoVim吧！​

nvim

后面的话​

你可能会觉得就这？
